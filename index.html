How To Play

Once you begin the game you will see a set of cards flipped downwards

You will need to select a card and it will show you a color

You will need to find where the other version of that card is on the board by selecting a second card

Once you get a match the cards will stay open as a matched set

Match all the cards to each other to move on to the next level
